ID: BUG-001
Название: Функция capitilize из класса StringUtils модуля string_utils не обрабатывает корректно строки, состоящие только из пробелов.

Описание:
Шаги для воспроизведения:
1. Вызвать функцию capitilize с аргументом, состоящим из строки, содержащей только пробелы: "    ".

Ожидаемый результат:
При передаче во входных данных функция выбрасывает исключение (например, ValueError) или возвращает явный индикатор ошибки, чтобы предотвратить некорректную обработку данных.

Фактический результат:
Функция возвращает входную строку без изменений, не сигнализируя об ошибке.

Серьезность: S4
Приоритет: Medium

Среда: 
ОС: Windows 10 Домашняя версия 1903, сборка 18362.836 
Python 3.13.3
Модуль string_utils

Комментарий:
Рекомендуется добавить проверку входных данных и выбрасывать исключение при некорректных значениях.
